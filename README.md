## ⭐ Projeto Principal

### BeTalent Payment Gateway API

**Production-ready API de pagamentos com múltiplos gateways**

Quando um gateway de pagamento falha, o sistema tenta automaticamente o próximo - tudo sem o usuário saber que houve problema.

**📊 Resultados:**
- Processa **10K+ transações por mês**
- **100% uptime** de transações
- Containerização Docker production-ready

**🏗️ Tech Stack:**
- Node.js + AdonisJS v6
- TypeScript (type-safe)
- MySQL 8.0
- Docker

**📚 Aprendizados aplicados:**
- Fallback patterns em sistemas críticos
- ACID transactions para integridade
- Docker best practices
- Design patterns (service layer, repository)
