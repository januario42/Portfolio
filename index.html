Sobre Mim

Olá! Sou um estudante de nível técnico com uma paixão profunda por desenvolvimento de software e gerenciamento de dados. Com uma sólida base em linguagens de programação como Java e Python, estou sempre em busca de novos desafios e oportunidades para expandir meus conhecimentos.

Seja resolvendo problemas complexos com código ou otimizando sistemas de dados, sou movido pela curiosidade e pela vontade de crescer na área de desenvolvimento de software.
